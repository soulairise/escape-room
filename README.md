# 🕰️ 사라진 시계공의 공방 (The Clockmaker's Room)

브라우저에서 즉시 플레이하는 방탈출 게임. 순수 HTML · CSS · JavaScript로만 만들어졌고, 외부 라이브러리나 오디오 파일 없이 동작합니다.

## 플레이

👉 **[여기서 플레이](https://soulairise.github.io/escape-room/)**

또는 이 저장소를 내려받아 `index.html`을 브라우저로 열면 됩니다.

## 특징

- **방 3개 연결** — 작업실 → 서재 → 시계탑 (자유롭게 이동)
- **이미지(SVG) 클릭형** — 배경과 오브젝트가 모두 코드로 그려짐
- **복합 퍼즐** — 아이템 획득 → 잠금 해제 → 단서 조합 → 암호 입력 → 시계 조작
- **스토리** — 사라진 스승 엘리아스를 찾는 견습생의 이야기가 화면 자막과 일지로 이어짐
- **오디오** — Web Audio로 만든 배경음악·효과음, 브라우저 내장 한국어 음성 내레이션(TTS)

## 조작

- 오브젝트를 클릭해 조사하고 아이템을 모으세요.
- 🎒 가방 / 📖 일지 / 💡 힌트 버튼을 활용하세요.
- 🔊 · 🗣️ 버튼으로 배경음악·효과음과 음성 내레이션을 켜고 끌 수 있습니다.

## 목표

단서를 모아 시계탑의 대형 괘종시계를 **스승이 사라진 시각**에 맞추면 탈출구가 열립니다. 3분 이내 클리어에 도전해 보세요!

## 파일 구성

| 파일 | 역할 |
|------|------|
| `index.html` | 게임 화면 구조 |
| `style.css` | 방·팝업·UI 디자인 |
| `game.js` | 상태 관리 · 퍼즐 · 판정 · 인벤토리 · 스토리 |
| `audio.js` | 배경음악 · 효과음 · 음성 내레이션 엔진 |
